## рџљЂ Getting Started

```bash
git clone https://github.com/arpan-1010/Ludo-King.git
cd Ludo-King
bun install
```

Create environment files for backend/database (DATABASE_URL, JWT_SECRET, etc.), then:

```bash
bun run dev
```

Run Prisma migrations:

```bash
bunx prisma migrate deploy
```
